## 🚀 How to Use

### Clone & Run
```bash
# Clone the repository
git clone https://github.com/aminchedo/Crepto_AiLastChance.git
cd Crepto_AiLastChance

# Navigate to HTS Trading System
cd hts-trading-system

# Start Backend
```
