🎯 USAGE SCENARIOS
=================================================================

SCENARIO 1: FIRST TIME SETUP
-------------------------------------------------------------------
Step 1: check-dependencies.bat
        (Verify Node.js, npm, etc. are installed)

Step 2: If all checks pass → start-app.bat
        (Starts both servers automatically)

SCENARIO 2: DAILY DEVELOPMENT
-------------------------------------------------------------------
Just run: start-app.bat

The script will:
✅ Check Node.js
✅ Stop old processes
✅ Start backend (port 5000)
✅ Wait for backend
✅ Start frontend (port 5173)
✅ Open browser

SCENARIO 3: NEED CONTROL OVER SERVERS
-------------------------------------------------------------------
Run: process-manager.bat

Interactive menu with options:
1. Start Both Servers
2. Stop All Servers
3. Restart Backend Only
4. Restart Frontend Only
5. Restart Both Servers
6. View Running Processes
7. Health Check
8. View Server Logs
9. Install Dependencies
0. Exit

SCENARIO 4: WORKING ON BACKEND ONLY
-------------------------------------------------------------------
Run: start-backend-only.bat

Backend runs on: http://localhost:5000

SCENARIO 5: WORKING ON FRONTEND ONLY
-------------------------------------------------------------------
Run: start-frontend-only.bat

Frontend runs on: http://localhost:5173
(Expects backend already running)

SCENARIO 6: PRODUCTION TESTING
-------------------------------------------------------------------
Run: start-production.bat

Process:
1. Cleans old builds
2. Runs npm run build
3. Starts production server
4. Opens browser

=================================================================
🔧 TROUBLESHOOTING
=================================================================

ISSUE: "Port already in use"
-------------------------------------------------------------------
SOLUTION 1: Use process manager
  process-manager.bat → Option 2 (Stop All Servers)

SOLUTION 2: Manual kill
  taskkill /f /im node.exe

ISSUE: "Node.js not found"
-------------------------------------------------------------------
SOLUTION:
1. Install from: https://nodejs.org/
2. Restart terminal
3. Run: check-dependencies.bat

ISSUE: "Dependencies missing"
-------------------------------------------------------------------
SOLUTION 1: Use process manager
  process-manager.bat → Option 9 (Install Dependencies)

SOLUTION 2: Manual install
  npm install
  cd backend && npm install

ISSUE: Servers won't start
-------------------------------------------------------------------
SOLUTION:
1. Run: check-dependencies.bat
2. Fix any ❌ errors shown
3. Run: start-app.bat again

ISSUE: Backend not responding
-------------------------------------------------------------------
SOLUTION:
  process-manager.bat → Option 3 (Restart Backend Only)

ISSUE: Frontend won't load
-------------------------------------------------------------------
SOLUTION:
  process-manager.bat → Option 4 (Restart Frontend Only)
